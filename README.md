# Tweeter
[![CircleCI](https://circleci.com/gh/Shayne3000/Tweeter.svg?style=shield)](https://circleci.com/gh/Shayne3000/Tweeter/)

Tweeter is a twitter clone written in Kotlin using modern android development techniques from the Jetpack suite. It employs industry best practices as it pertains to scalability, testability, accessibility, and maintainability.
